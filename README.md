# 💊 Drug Regulatory Classification

This project uses Machine Learning to classify drugs as Regulated or Non-Regulated based on clinical and risk-related features.

## 🔧 Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn, XGBoost
- Flask

## 📊 Features
- Data Preprocessing & EDA
- Model Comparison (Logistic Regression, Random Forest, XGBoost)
- Hyperparameter Tuning (GridSearchCV)
- Model Deployment using Flask

## 📈 Results
- Accuracy: 72%
- Recall: 75%
- F1-score: 0.58

## 🚀 How to Run
pip install -r requirements.txt
python app.py
